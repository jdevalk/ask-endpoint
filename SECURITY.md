# Security Policy

## Supported Versions

| Version | Supported          |
|---------|--------------------|
| 0.2.x   | Yes                |
| < 0.2   | No                 |

## Reporting a Vulnerability

If you discover a security vulnerability, please report it responsibly.

**Do not open a public issue.**

Instead, use [GitHub Private Vulnerability Reporting](https://github.com/jdevalk/ask-endpoint/security/advisories/new) to submit your report. You can expect an initial response within 48 hours.

Please include:

- A description of the vulnerability
- Steps to reproduce it
- The potential impact
- A suggested fix, if you have one
